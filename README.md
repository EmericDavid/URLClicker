# URL Clicker is a handy addon for World of Warcraft that enhances your chat experience by making **URLs clickable**. 

Say goodbye to the hassle of manually typing URL from chat messages. With URL Clicker, **simply click on any URL** shared in your chat and you can **directly copy it**. 

This addon automatically detects and formats URLs, allowing you to quickly access external links without interrupting your gameplay.  
  
If you have any bug, you can post a comment (with [BugSack](https://www.curseforge.com/wow/addons/bugsack) and [BugGrabber](https://www.curseforge.com/wow/addons/bug-grabber)) or add me on discord (mereci)
